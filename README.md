## AptDAG GP

```
install.packages(c("GPvecchia", "Rcpp", "RcppArmadillo", "RcppEigen"))

devtools::install_github("mkln/aptdag")
```
Or download source in zip file, then
```
devtools::install_local("aptdag-main.zip")
```
